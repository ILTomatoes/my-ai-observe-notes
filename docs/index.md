# AI 观察笔记

> 私人知识门户。收录 + 综合,经脱敏处理后公开。

## 关于本站

这是一个基于 MkDocs + GitHub Pages 的静态站,内容来自私有笔记仓库的精选。

- **收录原则**:经过人工挑选,值得公开分享的 AI/工程/认知类文章综合
- **脱敏处理**:所有发布内容已移除个人/公司/项目敏感信息
- **更新节奏**:不定期,基于平时积累

## 收录列表

参见[收录索引](articles.md)。

## 收录类型

- **单篇笔记**:一篇文章的原文要点 + 自己的观察
- **主题综合**:同一主题下多篇文章的串联综合,提炼跨篇主线

## 技术栈

- 静态站框架:[MkDocs](https://www.mkdocs.org/) + [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
- 部署:GitHub Actions + `actions/deploy-pages`
- 内容来源:私有 Git 仓库(经脱敏后人工挑选)
